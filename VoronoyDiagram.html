Como sabemos, para el diagrama de Voronoi nos hacen falta coordenadas. Por lo tanto, nos quedamos con la longitud, LON, y con la latitud, LAN. Empezamos borrando los valores nulos de estos campos:

In [3]:
# Importamos las librerias necesarias para el algoritmo Voronoi
import numpy as np
from scipy.spatial import Voronoi, voronoi_plot_2d
import matplotlib.pyplot as plt

# Necesitamos un array con la latitud y la longitud
puntos = data[['LON', 'LAT']].values

# Creamos el objeto Voronoi
voronoi = Voronoi(puntos)

# Dibujamos el diagrama
fig = voronoi_plot_2d(voronoi)
plt.show()

Para visualizar mejor el diagrama, vamos a tomar una pequea muestra de los datos. Adems, vamos pedirle que no nos muestre los vrtices, es decir, los puntos de interseccin entre las regiones (puntos naranjas) para que la visualizacin sea lo ms clara posible.

In [4]:
sample_data = data.sample(frac=0.00011, random_state=4)
# Creamos el array con la latitud y la longitud
puntos_sample = sample_data[['LON', 'LAT']].values
# Creamos el objeto Voronoi
voronoi_sample = Voronoi(puntos_sample)
# Dibujamos el diagrama
fig = voronoi_plot_2d(voronoi_sample, show_vertices=False)
# Lo coloreamos para distinguir mejor las regiones
for region in voronoi_sample.regions:
    if not -1 in region:
        polygon = [voronoi_sample.vertices[i] for i in region]
        plt.fill(*zip(*polygon))
plt.show()

Vemos algunas regiones en blanco porque son infinitas, es decir, en el diagrama anterior no estn definidas todas sus fronteras.
